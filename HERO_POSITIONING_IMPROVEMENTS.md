# Hero Positioning Improvements

## Current Weaknesses

The original hero section suffered from several messaging problems that made it feel generic and undifferentiated:

1. **"We build digital experiences that help businesses grow"** — This is the most common cliché in agency marketing. It says nothing unique about Massar. Every agency claims to "help businesses grow."

2. **Service-listing subtitle** — "We design and develop websites, applications, and brand identities" reads as a menu, not a value proposition. It fails to communicate *how* or *why* Massar is different.

3. **No strategic differentiation** — The website's "Why Us" section reveals real differentiators (strategy-first approach, craft obsession, reliability), but none of these appeared in the hero.

4. **Vague badge** — "Specialized digital studio" begs the question: specialized in what? It adds no clarity.

5. **No location signal** — Massar is based in Algeria, serving the local market, but the hero never mentions this. For local businesses, knowing the agency is based in Algeria builds immediate trust and relevance.

6. **No namedropping** — The agency has real clients (Journeya, DarLink) but the hero relied solely on anonymous metrics ("10+ Clients") instead of leveraging real brand names.

7. **No pain point addressed** — The hero never acknowledges why a visitor might be dissatisfied with their current digital presence or what specific problem Massar solves.

8. **Missing trust statement** — Credibility was reduced to three numbered metrics without narrative context.

## New Positioning

The new hero is built on a single, defensible positioning statement:

> **"Strategy-first digital products built for measurable growth."**

This positioning:

- **Leads with process** ("strategy-first") — Massar's real differentiator is that they start with goals and audience, not design. This sets them apart from agencies that jump straight to visuals.
- **Names the category** ("digital products") — broader than "websites," covers their full service range (web, mobile, branding, UI/UX, automation, AI).
- **Promises outcomes** ("measurable growth") — specific and accountable, replacing the vague "help businesses grow" with something that implies data, metrics, and results.
- **Establishes local relevance** — The badge "Digital studio — Algeria" immediately signals to local businesses that Massar understands the Algerian market.
- **Builds credibility with real names** — The trust statement "Trusted by Journeya, DarLink, and businesses across Algeria" uses verified client names instead of anonymous metrics.

## Design & Code Changes

### Component Changes (Hero.tsx)

- Added a new `<motion.p>` element between the CTA buttons and the metrics row displaying the trust statement.
- Added a new `<motion.div>` process row below the metrics row with three items (discovery, delivery, support) using the same Framer Motion `fadeUp` variant at `delay: 0.22`.
- Used consistent visual styling (`flex-wrap`, `gap-6`, `text-[13px] text-[#71717A]`, colored dot indicators) to match the existing trust metrics row design.
- Used `bg-sky-500` dots for the process row to visually distinguish it from the metrics row above (which uses emerald/purple/amber dots).
- Adjusted the trust metrics row spacing from `mt-8 sm:mt-10` to `mt-6` for tighter vertical rhythm after adding the trust statement.
- Used `mt-5` spacing for the new process row to maintain a clean hierarchy.
- No new dependencies or imports added.

### Locale Changes

- New `hero.trustStatement` key added to all three locales.
- New `hero.process` object with `discovery`, `delivery`, and `support` keys added to all three locales.
- Fixed duplicate `badge` key in `ar.json` (was duplicated on two consecutive lines).
- All existing keys preserved for backwards compatibility.

## Expected Business Impact

| Factor | Improvement |
|--------|-------------|
| **Clarity** | "Strategy-first" immediately communicates process differentiation. "Algeria" badge establishes local relevance. Visitors understand *who* this is for and *why* it's different within 3 seconds. |
| **Differentiation** | The new headline is specific and defensible. No competing agency can use it without also being strategy-first, outcome-driven, and Algeria-based. |
| **Trust** | Real client names (Journeya, DarLink) replace anonymous metrics as the primary credibility signal. The metrics row reinforces with quantitative proof. |
| **User engagement** | Visitors who identify as "businesses that care about strategy and results" (the target persona) will feel the messaging speaks directly to them, increasing time-on-site and scroll depth. |
| **Conversion** | The CTA "Start your project" is preserved (tested language), but now follows a stronger value proposition. Visitors arrive at the contact form with clearer expectations about Massar's approach and value. |
| **Uncertainty reduction** | The new process row ("Free discovery session • 2–12 week delivery • Ongoing support") answers three of the most common pre-purchase questions before the visitor has to scroll or ask: what's the process, how long does it take, and what happens after launch. This reduces anxiety and friction. |
| **SEO** | Keywords like "digital studio Algeria," "strategy-first digital products," and "measurable growth" are naturally integrated without stuffing. The badge "Digital studio — Algeria" reinforces local SEO relevance. |
| **Accessibility** | Clear `<h1>` heading, readable 13-16px body copy, descriptive CTA labels, and inclusive language maintained. New trust statement uses semantic `<p>` element. |
